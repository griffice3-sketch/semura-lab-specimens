# 検体001：トレンドフォロー型スキャルピングEA（Stochastic + EMA + ATR利確 + 固定SL）

> Semura Lab. EA構造診断アーカイブ / 検体カード

## 一行所見

利確2〜5pips・損切固定30pips（RR 1:6〜1:15）の利小損大型。高勝率に見えるバックテストの裏に、決済不発・未確定足参照（未来リーク）・スプレッド制限の空回りという複数の構造的欠陥を持つ「型」の代表例。

## 検体プロフィール（事実）

| 項目 | 内容 |
|------|------|
| 種別 | トレンドフォロー型スキャルピングEA（MQL4） |
| 時間軸 | M5（5分足）。トレンド判定にEMA204、加速判定にM1のEMA4 |
| シグナル | Stochastic(13,3,3) の過熱＋クロス → 短期EMA加速で追随エントリー |
| 利確(TP) | ATR可変：通常5pips / 低ボラ2pips |
| 損切(SL) | 固定30pips |
| リスクリワード | 1:6 〜 1:15（利小損大） |
| 補助 | スプレッド上限・時間帯・最小間隔・RSSニュース回避 |

## 診断サマリ（詳細はレポート本体へ）

| # | 深刻度 | 内容 |
|---|--------|------|
| A | FATAL | 決済条件が `(atr<x) AND (atr>x)` で恒偽。能動利確が一度も発火しない |
| B | FATAL | Stochastic/MA を shift=0（未確定足）参照。リペイント＝バックテスト幻覚 |
| C | FATAL | スプレッド制限の参照変数が未初期化（常に0）。フィルタが空回り |
| D | 重大 | Short の損益評価で Ask/Bid 取り違え。系統誤差 |
| E | 重大 | RR 1:6〜1:15 の利小損大。高勝率は損切りを遠くに置いた代償 |
